被动型生物

蝙蝠（本游戏第四小的生物）

生命值 6(★★★) 图片
体型 高度：0.9方格
宽度：0.5方格
生成于 63层以下
临近方块亮度低于4
10月20日至11月13日：临近方块亮度小于等于6
掉落物 无
经验值 0

鸡

生命值 4(★★) 图片
生成于 固体方块表面上方至少有两格的空间，通过投掷鸡蛋腐化（1/8的概率）
经验值 杀死成年鸡：1-3
繁殖：1-7

哞菇

哞菇：牛的独特变种

生命值 10(★★★★★) 图片
体型 成年个体：高度：1.4方块
宽度：0.9方块
幼年个体：高度：0.7方块
宽度：0.45方块
生成于 蘑菇岛
经验值 杀死成年哞菇：1-3
繁殖：1-7

猪

生命值 10(★★★★★) 图片
生成于 上方至少有2×2空间的非透明方块上
常见掉落物 生猪肉（1-3）未着火时
烤猪肉（1-3）着火时
可使用物品 鞍
经验值 杀死成年猪：1-3
繁殖：1-7

兔子

生命值 3(★★☆) 图片
护甲值 仅杀手兔：8
攻击力 仅杀手兔：
简单：5（★★☆）
普通：8（★★★★）
困难：12（★★★★★★）
常见掉落物 兔子皮
生兔肉（未着火时）
熟兔肉（着火时）
稀有掉落物 兔子脚
经验值 杀死成年兔：1-3
繁殖：1-4

马

生命值 15(★★★★☆)-30（★×15） 图片
体型 成年：
高：1.6方块
宽度：1.397方块
幼年：高：0.8方块
宽度：0.698方块
生成于 草原和热带草原
常见掉落物 皮革（正常）
骨头(骷髅马)
腐肉（僵尸马）
可使用物品 鞍
马铠
箱子
经验值 杀死成年马：1-3
繁殖:1-7

鱿鱼

生命值 10(★★★★★） 图片
生成于 至少一格水，任意亮度，出生所在方块高度必须在45（包含）到海平面之间
掉落物 墨囊
经验值 1-3

海龟

生命值 30(★×15) 图片
体型 成年：
高度：0.4方块
宽度：1.2方块
幼年：高度：0.12方块
宽度：0.36方块
生成于 温暖海滩
掉落物 海草
经验值 杀死成年海龟：1-3
繁殖：1-7

村民

生命值 20(★★★★★★★★★★) 图片
