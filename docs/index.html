NPC介绍
我的世界NPC主要分为：攻击型生物和传说生物、中立型,效用型与可驯服型生物、被动型生物。不同的生物有不同的属性，玩家可以根据它们的不同来感受不同的游戏体验
Minecraft 算法分析
Minecraft算法主要由地图生成函数、生物群系、爆炸算法等多种算法组成，不同的算法对应产生不同的视觉、听觉效果，现实感很强
Minecraft 各大论坛
通过Minecraft论坛可以与各位大神近距离接触，体验玩家互动时的快乐。同时也可以得到自己想要的材料，学习到各种骚操作，达到事半功倍的效果
